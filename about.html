About

Łukasz Kapuśniak. Online as kapoost. Warsaw, Poland.

The day job

I'm CTO at CloudTechnologies (onAudience), a Warsaw-based B2B data company. Most of my professional work happens around audience signals, identity-resolution plumbing, and the messy real surface where AI agents are starting to participate in the ad-tech stack as peers, not just consumers.

For the past several months a large share of that has been the Ad Context Protocol — the open spec for how agents discover inventory, finalise proposals, and execute media buys with publishers. I run a small reference seller agent against it on every release candidate, file the issues that the storyboard runner doesn't catch, and contribute back the schema fixes that close them.

The work outside

Public repositories on github.com/kapoost: a personal MCP server (humanmcp-go), a local RAG vault (mysloodsiewnia), a federated marketplace across humanMCP servers (humanmcp-marketplace), an AdCP signal-provider monorepo (adcp-signals), and a telescope kiosk that runs a procedural jazz engine while it stacks live frames (seestar-kiosk).
